# NITC-Person-Rescue
A UAV person Rescue dataset. Details of the dataset can be found in **Fine-tuned deep models for niche datasets â€” People detection in UAV building images to aid rescue operations**. Please cite this paper if you are using this dataset.
The dataset contains UAV images of people trapped inside building, roofs and balconies. Overall 9 videos are captured and around 2600 images are annotated for person class. Annotations consists of object detection labels and captions. 
The dataset can be downloaded through the link, https://drive.google.com/drive/folders/1D5SNPhMh04yc-MNDS0uzobiozXVLGqhx?usp=sharing . Please request access first, you can download it once access is granted.
